# Phase 6 Deliverables

Final-project report and presentation for CPTS 315 — Music Wrapped.

## Files

| File | Purpose |
|---|---|
| `main.tex` | IEEE conference-format LaTeX report. Compile with `pdflatex` + `bibtex`. |
| `references.bib` | Bibliography for `main.tex`. |
| `IEEEabrv.bib`, `IEEEtran.bst`, `ieeeconf.cls` | IEEE conference template assets (carried from `docs/template/`). |
| `presentation.pptx` | 12-slide presentation deck (16:9), generated by `build_pptx.py`. |
| `build_pptx.py` | Script that builds the deck from scratch using `python-pptx`. Re-run after editing. |
| `generate_figures.py` | Script that produces all 14 figures used in both the report and the deck. Re-run after model changes. |
| `figures/` | Output PNGs from `generate_figures.py`. |

## Compile the report (LaTeX)

```bash
cd docs/report
pdflatex main.tex
bibtex main
pdflatex main.tex
pdflatex main.tex
```

Or open `main.tex` in Overleaf and upload the IEEE class files alongside it.

## Rebuild the slide deck

```bash
python docs/report/build_pptx.py
# → docs/report/presentation.pptx
```

## Regenerate figures

Figures are derived from the live catalog + persisted artifact:

```bash
python docs/report/generate_figures.py
```

Requires that `data/high_popularity_spotify_data.csv`, `data/low_popularity_spotify_data.csv`, and `artifacts/personality_v1.joblib` exist. Run `notebooks/02_clustering.ipynb` first if the artifact is missing.

## Deck design

- **Palette:** Midnight Executive with a Spotify-green accent (`#1DB954`). Dark slides for the title, archetype showcase, and closing; light slides for content.
- **Layout:** 16:9 widescreen (13.33" × 7.5"), 0.6" page margins.
- **Typography:** Calibri throughout with bold 32pt titles, 14–18pt body, 10–11pt captions.
- **Visual motif:** thin green accent bar on every section title; numbered green circles for ordered lists.

## What's in the report (mapping to plan §6 deliverables)

- §1–§4 of the .tex: project framing, problem formulation, methodology — ties directly to Phases 1–5.
- §5: experiments and results — leave-one-out, alpha sweep, cosine vs Euclidean, ILD, hidden-gem disjointness, cluster stability.
- Limitations + future work close out Phase 6.

The deck mirrors the report sections with one slide per major beat plus a dedicated archetypes showcase.
